## 🚀 Como Importar/Executar

### No Eclipse:
1. **File** → **Import** → **Existing Projects into Workspace**
2. Selecione a pasta `exercicio-java`
3. Marque os projetos desejados
4. Clique em **Finish**
5. Execute com botão direito na classe → **Run As** → **Java Application**
